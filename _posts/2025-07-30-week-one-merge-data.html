Both datasets have a common identifier (id) named CountryCode. We will use the merge command and the common id to merge the two datasets on a one-to-one basis.

* Load Constant_GDP dataset in Stata
use "./Stata/Constant_GDP.dta", clear        

* Merge CountryCode.dta dataset with dataset Constant_GDP dataset
merge 1:1 CountryCode using "./Stata/Country_Group.dta"

The two datasets are now merged. The result shows that all 217 observations matched between the two datasets.
